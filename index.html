How it's done?

Added some inline style to override so the ripple is fully visible.

.override:after {
  background-color: red;
}

Usage

To enable this feature just include CSS file and put "ripple" class on HTML element. The only problem is that it doesn't follow mouse location because it is not possible via CSS, so it always starts somewhere towards bottom-left region.
