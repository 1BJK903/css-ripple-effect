How it's done?

Added some inline style to override so the ripple is fully visible. It is not leaving the block when by default.

.override {
  overflow: visible;
}
.override:after {
  background-color: red;
}

Usage

This feature is currently added to ".btn" and ".navbar a" selectors. It can be added as a class or any other selector. The only problem is that it doesn't follow mouse location because it is not possible via CSS, so it always starts somewhere towards bottom-left region.
